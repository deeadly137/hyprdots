Public repository to save my current hyprland dotfiles

You can use it if you want, it's nothing but a caelestia shell edited dotfiles.

If you're interested in caelestia shell:
https://github.com/caelestia-dots/shell (https://aur.archlinux.org/packages/caelestia-shell)

to set up caelestia shell, do this in your terminal:
```
## NOTE THAT USING THIS COMMAND WILL WIPE YOUR CURRENT DOTFILES (/home/$USER$/.config/hypr/)
## The following command will ask you to install tools such as:

caelestia install

```
